S2Wicketが提供するUIコンポーネント構築機構のためのコアクラスを持つパッケージです。

このパッケージには，S2Wicketが提供するUIコンポーネント機構を実現するためのクラスが収められています。 開発者は，このパッケージにあるクラスを利用することにより，S2Wicketの機能を有効にすることができます。

S2Wicketが提供するUIコンポーネント構築機構とは，Wicketによって管理されるUIのためのコンポーネントツリーの構築処理について， 簡潔に記述するための各種アノテーションの提供と，各種処理をOGNL式で記述し実行するための機構の提供から成り立ちます。

ComponentクラスのオブジェクトがWicketに登録された際のコールバックをアプリケーションが受けるために， wicket.application.IComponentInstantiationListenerインタフェースが提供されています。 S2Wicketは，このIComponentInstantiationListenerインタフェースの実装クラスとして， {@link org.seasar.wicket.uifactory.WicketComponentBuildingListener}クラスを提供します。WicketComponentBuildingListener クラスのインスタンスを生成し，wicket.Application#addComponentInstantiationListener()メソッドを 使用してWicketに登録することで，S2Wicketの機能が有効になります。

コンポーネントの生成は，{@link org.seasar.wicket.uifactory.WicketComponent}アノテーションを使用することで， S2Wicketに指示をすることができます。また，コンポーネントに関連付けるモデルの生成は， {@link org.seasar.wicket.uifactory.WicketModel}アノテーションを使用することで， S2Wicketに指示をすることができます。
